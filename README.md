# Blood-Bank-Management-System
The Blood Bank Management System is a web-based platform that connects donors, hospitals, and patients for efficient blood management. Built with HTML, CSS, JavaScript it enables donor registration, hospital management, and blood requests, ensuring a reliable, user-friendly, and accessible donation process.
